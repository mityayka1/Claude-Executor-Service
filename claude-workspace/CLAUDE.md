# PKG AI Tasks Workspace

Окружение для выполнения AI-задач проекта PKG (Personal Knowledge Graph).

## Контекст

PKG — система для интеллектуального хранения и извлечения контекста
взаимодействий с людьми и организациями.

**Цель:** Перед любым взаимодействием получить компактный контекст:
кто это, о чём договаривались, какие открытые вопросы.

## Общие правила

1. Возвращать только JSON согласно переданной схеме
2. Не выдумывать факты — только явно указанные в тексте
3. Русский язык для русскоязычного контента
4. Краткость и точность
5. При отсутствии данных — пустые массивы/null

## Доступные агенты

| Агент | Описание | Модель |
|-------|----------|--------|
| `fact-extractor` | Извлечение фактов из сообщений | haiku |
| `summarizer` | Создание резюме переписок | sonnet |
| `profile-aggregator` | Агрегация профиля отношений | sonnet |
| `context-synthesizer` | Синтез контекста | sonnet |
